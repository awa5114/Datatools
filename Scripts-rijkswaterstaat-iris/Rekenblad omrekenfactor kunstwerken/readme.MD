# Doel

Excel-rekenblad waarmee voor kunstwerken de gemiddelde waarde van de sterkte kan worden bepaald. 

# Gebruik en functionaliteit

Rekenblad om de door de toetser afgeleide karakteristieke waarde van de sterkte van een constructie om te rekenen naar de gemiddelde waarde van de sterkte met behulp van de variatiecoëfficiënt van het materiaal van deze constructie. Deze gemiddelde waarde van de sterkte kan vervolgens samen met de variatiecoëfficiënt worden ingevoerd in Riskeer, zodat een probabilistische analyse kan worden gemaakt. In het eerste tabblad van de Excel is een uitgebreidere toelichting opgenomen. 

# Technische informatie

Betreft een Exel rekenblad.
